## Run Locally

Clone the project

```bash
  git clone https://github.com/Soufianetyk1337/cryptocurrency-app-with-react-redux-toolkit.git 
```
Go to the project directory

```bash
  cd cryptocurrency-app-with-react-redux-toolkit
```
Install the dependencies
```bash
  yarn install 
```

Start the server

```bash
  yarn start 
```
